    Life-Hacks!

    Learn some usefull lifehacks :D (for programmers)

    3 simple life hacks!

    #1 Keep your tabs clean with ONETAB

    Ever feel like you're building tabs constantly while doing a Project?

    A Chrome extention called ONETAB allows you to keep all your tabs in one page with ONE simple click!

    Bonus: Saves up to 95% memory

    #2 Use GITHUB as SOTRAGE

    Did you know you can store more then just code on Github, all types of documents like files, photos and ect.

    However there are some limitations that follow:

    • 100GB storage per repository
    • 100MB per individual file

    #3 Eat the frog

    No this does not mean literally eat a frog. As developers we tend to do the easiest things first which tends to be unproductive.

    what you want to do is use your brainpower for the hardest task at hand.

    leave the easier task for the end
